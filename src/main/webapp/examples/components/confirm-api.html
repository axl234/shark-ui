> 配置项： >> animate - 是否必须：否 - 类型：string - 默认值：'fade' - 说明：需要的动画效果 >> size - 是否必须：否 - 类型：string - 默认值：'' - 说明：弹窗尺寸 >> backdrop - 是否必须：否 - 类型：string - 默认值：'' - 说明：点击背景是否消失：static不消失 | ''消失 >> content - 是否必须：是 - 类型：string - 默认值：'' - 说明：弹窗主体内容 >> onShow - 是否必须：否 - 类型：function - 默认值：function() {} - 说明：modal显示时的回调函数 >> onHide - 是否必须：否 - 类型：function - 默认值：function() {} - 说明：modal隐藏时的回调函数
