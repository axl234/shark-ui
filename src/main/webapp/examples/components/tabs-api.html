> 配置项： >> event - 是否必须：否 - 类型：string - 默认值：'click' - 说明：触发事件，click|hover >> tabs - 是否必须：否 - 类型：array - 默认值：null - 说明：tab页的分页信息，eg：[{title: '前端组',pane: '都是小哥'},{title: '视觉组',pane: '都是大姐'}] 当使用 tabs = $.fn.sharkTabs(options) 方式生成tabs对象时，必须填写。 >> auto - 是否必须：否 - 类型：number - 默认值：0 - 说明：设置自动播放的时间（0为不自动播放） >> active - 是否必须：否 - 类型：number - 默认值：0 - 说明：当前是第几个tab >> onTabSwitch - 是否必须：否 - 类型：function - 默认值：function(index) {} - 说明：tab切换时触发的函数
